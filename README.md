## Commands
- `Spawn the Evil Entity`: Opens the Evil Entity panel
- `Delete the Evil Entity`: Closes the Evil Entity panel

## Notes
- The entity tracks your mouse and reacts to movement
- It runs in a seperate panel of its own
- If it gets annoying, use the stop command
